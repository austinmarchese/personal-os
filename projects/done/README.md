# Done Projects

Finished projects live here. They become training data for future work.

## Why This Matters

When you complete a project and move it here, Claude can reference it when you start similar work:

- "Last time you did X, you learned Y"
- "Your social media campaign used this structure"
- "You prefer this format for proposals"

## How to Use

When a project is complete, move it:

```bash
mv projects/my-project projects/done/my-project
```

That's it. It's now part of your training data.
