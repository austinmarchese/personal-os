# Best Work

Examples of YOUR best outputs. AI learns your standards from these, not from rules.

## Why This Matters

When you say "write like me," AI needs examples of what that means.

This folder holds your best:
- Emails that landed
- Reports that impressed
- Proposals that closed
- Writing you're proud of

## How to Use

### Add your best work

Organize by type:
```
best-work/
├── emails/
├── reports/
├── proposals/
├── writing/
└── [whatever categories make sense for you]
```

### Reference in prompts

When asking AI to write something:

"Write this email in my voice. Reference the examples in `context/best-work/emails/` for my style."

## Tips

- **Quality over quantity.** 5 great examples beat 50 mediocre ones.
- **Include context.** Add a note about why each piece worked.
- **Update regularly.** As you create new best work, add it here.
